Installation & Activation


🛠 Installation:

  1. Download from App Store/Play Store
  2. Enter Internet Banking User ID
  3. Enter 9-digit Activation Code
  4. SMS Verification from RMN
  5. Set 5-digit Login Passcode

💡Activation:

  1. Modes of Activation:
    • Debit Card (PIN)
    • Aadhaar OTP
    • Branch Activation Code
  2. Login with 5-digit Passcode
  3. Go to 'Activate Mobile Services'
  4. Set 6-digit MPIN
  5. Choose Activation Mode
  6. Validate & Start Banking
